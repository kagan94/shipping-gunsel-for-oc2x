Доставка "Гюнсел" по Украине для OpenCart 2.x.

Возможности:
- Выбрать географическую зону для доставки
- Выбрать порядок сортировки
- Задать фиксированную стоимость доставки
- Задать минимальную сумму заказа для бесплатной доставки
- Сделать доставку "Гюнсел" бесплатной
- Добавлена поддержка OpenCart 2.2.0.0

Инструкция по установке:
1) Загрузите папку "upload" в корневой каталог вашего сайта.
2) Активируйте доставку "Гюнсел" в разделе "Доставки"
